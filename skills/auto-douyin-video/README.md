# Auto Douyin Video 🎬

全自动的抖音视频生成与发布工作流。本项目结合了 Google Veo 3.1 视频大模型与 Playwright 自动化测试框架，实现“一句话剧本 -> 云端视频渲染 -> 本地下载 -> 抖音创作者中心全自动发布”的端到端无人值守流程。

## ✨ 核心特性

- 🤖 **AI 视频生成**: 接入 Google `veo-3.1-generate-preview` 长效生成接口，全自动轮询获取云端视频。
- 🛡️ **反反爬与弹窗克星**: 针对抖音网页版的各种新手指引、封面推广弹窗（如“设置横封面获取流量”）进行物理级精准拦截，绝不卡死流程。
- ⌨️ **物理级拟人输入**: 摒弃会被抖音风控识别的 `.fill()` 和易诱发退出 BUG 的 `Enter` 键。采用纯物理键盘 `keyboard.type` 结合 `Space` 键输入文案，完美生成蓝色话题实体标签。
- 🖼️ **智能封面处理**: 自动选择 `3:4` 竖封面并截取第一帧，遇到异常时自动降级点击“AI智能推荐封面”。

## 🚀 部署与安装指南

### 1. 环境准备
- **Node.js**: v18 及以上版本。
- **Playwright**: 用于底层浏览器自动化。
- **【免登录核心依赖】**: 本脚本运行基于本地持久化的浏览器用户目录。必须确保 `~/.agents/skills/douyin-creator-tools/.playwright/douyin-profile` 目录下已有登录过抖音的缓存数据（账号处于登录态）。

### 2. 克隆与安装依赖
```bash
git clone https://github.com/a23200/auto-douyin-video.git
cd auto-douyin-video
npm install
# 安装所需浏览器的内核依赖
npx playwright install chromium
```

### 3. 配置核心 API Key
在项目根目录下创建一个名为 `.env` 的文件，填入你申请的 Google AI Studio / Vertex AI 的 API Key：
```env
# 你的专属 Google Veo 3.1 API 密钥
VEO3_API_KEY=AIzaSyDxe...
```

## 🛠️ 使用方法与操作指南

工作流分为两步走（视频生成 -> 自动发布）。

### 步骤一：一键生成视频
向 `generate-video.js` 传入带有丰富分镜和画面的 Prompt 剧本。
```bash
node scripts/generate-video.js "电影级镜头，一位白须飘飘的中国老者，身着飘逸的传统汉服，在古风木制凉亭内的石桌旁优雅落座。他轻轻端起青瓷茶杯细细品茗。清晨柔和的阳光..."
```
> **提示**：云端渲染需要大约 1~3 分钟，脚本会自动呈现轮询进度。
> **结果**：执行完毕后，视频将被下载到项目根目录（例如：`output_video_1774083669834.mp4`）。

### 步骤二：自动化发布抖音
拿着第一步生成的视频绝对路径，以及你想发送的抖音文案，传给 `publish-video.mjs`。
```bash
# 请注意将路径和文案替换成自己的真实内容
node scripts/publish-video.mjs "/Users/xxxx/auto-douyin-video/output_video_1774083669834.mp4" "偷得浮生半日闲🍵。岁月静好，一杯清茶品尽人生百味。在这个快节奏的时代，偶尔也需要让心静下来。#古风 #茶道"
```
> **自动发布流程说明**：
> 1. 脚本将隐式拉起本地绑定的抖音号 Chrome 窗口。
> 2. 自动上传视频并强制等待 `20` 秒云端解析。
> 3. 清理各类新人指导弹窗，进入封面设置界面，选择 `3:4` 竖版完成截取。
> 4. 遭遇次生弹窗（横版封面推广）时自动点击“暂不设置”放行。
> 5. 以真人打字速度敲入文案，并将 `#古风` 等格式转换为真实的蓝色话题标签。
> 6. 精准寻找到底部的“发布”按钮进行发射。
> 
> **安全机制**：执行结束（无论成功或报错），浏览器都会停留在桌面上 **5分钟** 供你人工检阅发布结果，之后才会自动销毁释放资源。

# robot_vlm

基于 `vlm_arm` 思路为本机机械臂单独新建的一套控制目录。

## 目标
- 不改现有 `agents/robotics/`
- 单独孵化一套“自然语言 -> 视觉定位 -> 像素转机械臂坐标 -> 执行动作”的新流程
- 优先适配我们当前的 OpenClaw + pymycobot 工作方式

## 当前目录
- `config/`：标定参数、设备参数、环境变量模板
- `prompts/`：LLM / VLM 系统提示词
- `temp/`：运行时临时文件
- `logs/`：日志
- `docs/`：设计文档
- `tests/`：测试脚本

## 规划
第一阶段先只把流程、提示词、配置和测试骨架搭起来。
第二阶段再接真实摄像头、真实机械臂、真实 VLM。

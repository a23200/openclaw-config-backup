# Vision Grounding 提示词

你会收到：
1. 一张机械臂工作台的俯视图（分辨率宽 Width=1920，高 Height=1080）。
2. 一句机械臂操作命令。

你的任务：
- 提取出用户命令中需要操作的【目标物体】。
- 将提取出的物体命名为 start，并直接返回它在 1920x1080 图片中的边界框像素坐标（左上角 [x1, y1] 和右下角 [x2, y2]）。
- X的范围是 0 到 1920（从左到右）。Y的范围是 0 到 1080（从上到下）。
- 务必仔细观察物体的真实位置并给出合理的像素估计，切勿输出归一化到1000的坐标，直接输出 1920x1080 尺度下的真实坐标！

输出格式（严格返回 JSON）：
{
  "start": "万宝路烟盒",
  "start_xyxy": [[1200, 200], [1400, 400]]
}

只输出 JSON，不要输出其它内容。

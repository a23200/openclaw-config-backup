# 轻约到店

这是由需求自动生成的 Web App 项目包。

## 你会得到什么

- 自动生成的多屏页面预览
- 本地可用功能：时段选择 + 本地预约记录
- 可安装的 PWA 配置（manifest + service worker）
- 业务功能配置 JSON

## 如何使用

1. 将整个目录部署到任意支持 HTTPS 的静态托管平台。
2. 用手机浏览器打开页面。
3. 如需 Web 形态运行，可在浏览器菜单中选择“添加到主屏幕”。

## 本地功能说明

- 用户可以直接在 App 里选择时段并生成本地预约记录，不需要后台也能演示流程。
- 点击时段即可预约
- 预约记录自动保存在本机
- 适合门店、咨询、服务预约

## 当前业务摘要

- 产品定位：围绕时间与服务资源编排，适合到店、咨询、医疗、美业等业务。
- 目标用户：需要预约服务的用户
- 核心卖点：减少人工沟通，提升到店与履约效率
- 盈利方式：预约成交、增值套餐、复购提醒

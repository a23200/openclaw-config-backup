# Generated Assets Archive

本目录用于归档此前由 PPT / 图片样张脚本生成的输出文件，避免散落在工程目录和桌面根目录。

- `runtime-image-ppt-demo/`：原 `runtime/image-ppt-demo` 下的图片型 PPT 样张、PNG 页面和 contact sheet。
- `desktop-generated/`：桌面根目录中明确由本仓库脚本输出的 PPT / PNG / 提示词文件。
- `MOVED_FILES.txt`：本次从桌面根目录移动过来的原始路径清单。

源码、测试文件、README、docs/assets、用户私人文档未移动。

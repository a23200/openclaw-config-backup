
### 音乐控制 (LX Music)

当用户说以下话时，请调用相应工具：

| 用户说的话 | 调用的工具 | 说明 |
|-----------|-----------|------|
| "打开音乐" / "播放音乐" | `open_music_player` | 打开LX Music播放器 |
| "播放" / "暂停" | `music_play_pause` | 播放/暂停当前音乐 |
| "下一首" / "切歌" | `music_next` | 切换到下一首 |
| "上一首" | `music_prev` | 切换到上一首 |
| "搜索音乐 xxx" | `music_search` | 搜索并播放指定音乐 |

**注意**: 用户说"放首歌"、"播放xxx"时，如果没有指定歌曲名，先调用 `music_search` 搜索，然后调用 `music_play_pause` 播放。

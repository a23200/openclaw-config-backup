# 设计观察与决策

## 总体判断
目标页属于典型的高校官网“纪念性信息页”风格：顶部是大图横幅 + 深色导航，中段是轻渐变二级栏目带，主体为高密度长篇正文，底部用波浪形深蓝页脚收束。页面的关键不是复杂交互，而是通过稳定秩序、庄重配色和长文阅读节奏传达机构权威感。

## 提炼出的设计语言

### 1. 主色与辅色
- 主色：深蓝、靛蓝系
  - 负责导航、标题、页脚、选中态和结构骨架
  - 传达“学术、权威、稳定、正式”
- 辅色：金棕、米金系
  - 只用于细节强调、标语、装饰线和波形层次
  - 作用像“校史感”与“纪念册感”的温度来源
- 背景：极浅灰蓝 + 暖白
  - 保证大段正文阅读舒适
  - 让深蓝和金棕更显克制

### 2. 字体层级
- 标题偏宋/衬线气质，更有机构历史感
- 导航和辅助信息字号较小，但字距拉开，显得正式
- 正文段落字号不大，但行高明显偏高，形成“长文也能顺畅读”的节奏
- 整体不是互联网产品那种强对比大字，而是偏文献、画册、年鉴式层级

### 3. 间距韵律
- 顶部横幅高度大，标题下沉明显，强调纪念感
- 中段栏目带与正文之间留有较大空白，形成章节切换感
- 正文容器宽而不满，行长受控，段落连续但不拥挤
- 页脚前后有明显纵向留白，让结尾更有收束仪式

### 4. 组件风格
- 一级导航：透明叠在横幅图上，强调站点级入口
- 二级导航：圆角胶囊 + 蓝到白渐变，是页面最鲜明的 UI 特征之一
- 正文容器：轻阴影、浅底、弱边界，不做强卡片感
- 页脚：波浪曲线 + 深蓝底，带一点文化场馆或校庆专题页气质

## 这次复刻的实现策略
- 不抓取原站图片/CSS/JS
- 自己手写一个“校园建筑与山体”的 SVG 背景插画，用来模拟横幅气氛
- 保留“高校门户”的信息结构，但文案全部改写为风格说明和抽象介绍
- 用 HTML + 内联 CSS + Tailwind CDN 快速完成结构和响应式
- 用低透明度建筑线稿和波浪页脚来强化“同一位设计师”的感觉

## 刻意不照搬的地方
- 没有复制原站真实 logo、图片、学校名称排版细节
- 没有逐字复写原站正文内容
- 没有照搬原导航的全部栏目与链接结构
- 没有复用原站资源地址或类名命名方式

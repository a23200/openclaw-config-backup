# 仿制设计笔记：cnblogs.com 风格提炼

## 整体观察
博客园首页的核心气质不是“营销型门户”，而是**以内容阅读为中心的技术社区首页**。视觉设计非常克制，重点放在高密度信息组织、稳定的阅读节奏和极低的学习成本上。

## 设计语言拆解

### 1. 主色与辅助色
- **主色**：高识别度的中高饱和蓝，接近 `#0f7ddf`
- **辅助色**：浅灰背景 `#f3f5f7`、白色卡片底 `#ffffff`
- **强调色**：少量橙色 / 红色用于“最多推荐”“新闻头条”等标签
- **文本色**：标题深灰接近 `#1f2937`，摘要与说明文字使用 `#6b7280` / `#9aa4b2`

### 2. 字体层级
- 顶部导航与侧栏：`13px - 14px`
- 榜单与摘要：`13px - 14px`
- 文章标题：`20px - 21px`，字重较高
- 元信息：`12px`

这种层级让页面虽信息密集，但不会显得吵。

### 3. 间距韵律
- 页面采用非常稳定的卡片式垂直节奏
- 模块间距多为 `16px - 20px`
- 单篇文章内部：标题、摘要、元信息之间形成规律性的纵向呼吸感
- 列布局间距明显，但不过度拉开，符合门户式信息组织方式

### 4. 组件风格
- **顶部导航**：白底、细边框、非常轻的层次，不靠重阴影抢视觉
- **左侧菜单**：激活项用蓝色文字 + 淡蓝底 + 左侧竖条强调
- **文章列表**：以分隔线为主，不依赖重卡片感；hover 时只做很轻的背景变化
- **右侧榜单**：标题 + 排名列表，结构极简，强调“扫读效率”
- **分页器**：矩形按钮、边框清楚、当前页使用主色填充

## 我这次的实现决策
- 没有照搬博客园源码、资源或图片
- 用 **HTML + 内联 CSS + Tailwind CDN** 重新组织了结构
- 保留了博客园最有辨识度的三栏布局和“信息流 + 榜单”的内容关系
- 把品牌替换为 `DevGarden`，避免视觉上过于直接复制
- 让卡片边框和阴影比原站稍微现代一点，但仍保持克制，不做花哨效果
- 图标和头像全部简化处理，避免依赖外部资源镜像

## 最终目标
这个版本追求的是：
**“像同一个设计师在 2026 年重新做了一版博客园首页”**，而不是像素级复刻。它保留了原站的阅读气质、内容密度和工程师社区味道，但代码结构更干净，组件命名也更清晰。

# cnblogs.com 风格仿制说明

## 视觉观察
博客园首页的设计核心是：**内容密度高、信息组织清晰、视觉语气克制**。它不是依靠大图或强交互动效来吸引点击，而是依靠标题、摘要、榜单和分类形成稳定的信息入口。

## 设计语言提炼

### 主色 / 辅色
- 主色：偏亮的技术蓝，接近 `#0b7adf`
- 页面背景：浅灰 `#f4f6f8`
- 卡片背景：纯白 `#ffffff`
- 辅助强调色：橙色、红色、青绿色，主要用于推荐标签

### 字体层级
- 导航 / 榜单 / 边栏：`13px - 14px`
- 元信息：`12px`
- 摘要：`14px`
- 文章标题：`21px` 左右，字重较高

### 间距韵律
- 三栏布局之间留白明显，但不过宽
- 卡片模块上下留白稳定，整体节奏偏理性、规整
- 文章项内部通常是：标题 → 摘要 → 元信息，三段式节奏非常稳定

### 组件风格
- 顶部导航：白底 + 细边框，轻微悬浮感
- 左侧菜单：激活项用蓝色高亮和浅蓝背景，不做厚重装饰
- 中央文章流：主要依赖分隔线和轻 hover，卡片感存在但不强
- 右侧榜单：极简列表，强调“扫读效率”
- 分页：矩形边框按钮，当前页用主色填充

## 本次实现决策
- 重新设计了品牌名 `CodeYard`，避免直接照搬原站视觉识别
- 保留博客园最重要的三栏阅读结构和技术社区信息气质
- 使用 HTML + 内联 CSS + Tailwind CDN 实现，结构更干净、可读性更高
- 没有抓取或复用目标站图片、CSS、JS、SVG 资源
- 让整体视觉略微现代化，但仍保持“工程师社区首页”的克制风格

## 目标
这份仿制稿追求的是：
**像博客园同风格体系下的一版现代化重绘**，而不是像素级复制。

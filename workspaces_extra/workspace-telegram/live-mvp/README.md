# Live MVP

一个本地可运行的直播 Web MVP，当前已经扩成一个更像直播平台原型的版本：

- 主播开播
- 观众输入房间号观看
- 直播间列表
- 聊天区
- 主播标题 / 封面
- 简单登录（昵称）
- 更接近短视频/直播平台风格的 UI

## 运行

```bash
cd live-mvp
npm install
npm start
```

打开：

- 主播端：`http://localhost:3000`
- 观众端：另一个浏览器 / 无痕窗口 / 另一台同网设备打开同地址

## 当前功能

### 已实现
- 单主播直播
- 多观众观看
- WebRTC 实时音视频
- Socket.IO 信令
- 聊天区
- 直播间列表
- 标题 / 封面
- 简单昵称登录
- 首页 + 房间感更强的 UI

### 暂未实现
- 真正账号系统
- 数据库存储
- 消息持久化
- 录播
- 礼物 / 弹幕 / 打赏
- 推荐算法
- 生产级 TURN / STUN 集群
- 审核 / 风控 / 管理后台

## 使用

1. 输入昵称
2. 输入房间号
3. 如果你是主播，再填直播标题、封面 URL（可留空）
4. 主播点击“我是主播，开始直播”
5. 观众点击“我是观众，进入房间”
6. 进入后可以在聊天区发消息
7. 左侧直播间列表可以直接切换想看的房间
